# Ctkrocks' Data Packs

## [Echo Horn](https://modrinth.com/datapack/echo-horn) v1.1
- Use an Echo Shard to upgrade a Goat Horn to perform sonic boom attacks.
- Upgrade a Goat Horn with an Echo Shard in the smithing table using the Ward Trim.
- The horn is given a durability of 50.
- The sonic boom can hit enemies through walls.

## [Echo Spyglass](https://modrinth.com/datapack/echo-spyglass) v1.0
- Updated for Minecraft: Java Edition v1.21.5.
- Locate vibrations through walls with an Echo Shard Spyglass.
- Use the Echo Spyglass to locate mobs through walls.

## [Echo Trim](https://modrinth.com/datapack/echo-trim) v1.1
- Updated for Minecraft: Java Edition v1.21.5.
- Adds an Echo Shard trim to armor via the Smithing Table.
- Applies a dark blue trim to your armor that has a subtle pulsing animation similar to sculk.

## [Prehistoric Mobs](https://modrinth.com/datapack/prehistoric-mobs) v1.0
- Updated for Minecraft: Java Edition v1.21.5.
- Adds the Big Beak (a large rideable bird), and the Hungry Dripleaf (a carnivorous plant).
- In trail ruins, you can find a Big Beak egg.
- Tame the Big Beak to ride it. It will slowly glide to ground, and can maneuver in-air.
- Sniffers can find this plant in lush caves.
- The Hungry Dripleaf will poison any mob standing on it. If the mob dies, the dripleaf will grow.
- Sneak to avoid the Hungry Dripleaf's attack.

## [Nautilus](https://modrinth.com/datapack/nautilus) v1.1
- Updated for Minecraft: Java Edition v1.21.5.
- Adds a Nautilus mob that hides in suspicious gravel underwater.
- The Nautilus is found in the suspicious gravel of cold ocean ruins.
- When found, the Nautilus will look for another gravel block to hide in.
- Kill a Nautilus to get its shell.

## [Biome Sniffer](https://modrinth.com/datapack/locate-biome-sniffer) v1.1
- Updated to add Pale Garden, Mangrove Swamp, and Cherry Grove.
- Removed Desert and Jungle.
- Use the Sniffer to locate biomes.
- Right click on the sniffer's face with one of these items, and it will sniff out the related biome.
- Many of these items can be obtained from a Wandering Trader.

| Biome | Item |
| --- | --- |
| Badlands | Red Sand |
| Dark Forest | Any Mushroom or Dark Oak Sapling |
| Flower Forest | Any Flower from a Flower Forest |
| Swamp | Lilypad |
| Pale Garden | Eyeblossom or Pale Moss or Pale Oak Sapling |
| Mangrove Swamp | Mangrove Propagule |
| Cherry Grove | Cherry Sapling |

## [Copper Pipes](https://modrinth.com/datapack/copper-pipes) v1.1
- Updated for Minecraft: Java Edition v1.21.5.
- Fill copper grates with water to create pipes.
- Dispense items into the grates to transport them around.
- Use bubble columns to push/pull items up/down the pipes. Downward bubble columns will pull items into the pipe.
- Use a campfire to fill the grates with smoke.
- The grates will heat up, damaging any mobs standing on them.
- This can be combined with water to create pipes that collect dropped items in mob farms.

## [Biome Vote](https://modrinth.com/datapack/biome-vote) v1.0
- Meerkats and Oases in the desert.
- Meerkat Burrows are a new structure found in deserts. A few Meerkats will spawn around each burrow.
- When it sees a hostile mob, the Meerkat will burrow in the sand.
- They'll check every few seconds for danger and pop back out if it's clear.
- Find an Oasis in the desert for a source of fresh water, plants, and the occasional Palm Tree.

## [Villager Caravan](https://modrinth.com/datapack/villager-caravan) v1.0
- Transport Villagers with Llamas by giving them the lead.
- Use a lead on a Llama, then sneak and right click on a Villager. The Villager will then start riding the Llama, and the lead will drop.
- Lead one Llama and the rest of the pack will follow.

## [Copper Mace](https://modrinth.com/datapack/copper-mace) v1.1
- Updated for Minecraft: Java Edition v1.21.5.
- Use a Lightning Rod to create a Copper Mace with a lightning attack.
- Craft the Copper Mace with a Lightning Rod and a Heavy Core.
- When holding the Copper Mace during a thunderstorm, it has a chance to be charged.
- The Heavy Core on the mace will light up to indicate its charged state.
- Each charge will give 4 lightning strike attacks.
- Hit any mob with the Copper Mace when charged to deal a lightning strike attack.
- All nearby mobs will be struck by lightning.
- The Copper Mace and Wind Burst enchantment are incompatible.
